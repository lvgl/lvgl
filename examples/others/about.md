This section groups auxiliary modules that do not belong to a single widget category: screen navigation helpers, the observer/subject binding system, the file explorer, translation tables, monkey test input, gesture recognizers, and similar utilities. Each page is self-contained and can be enabled in `lv_conf.h` independently of the others.
